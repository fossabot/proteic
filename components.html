Features

Proteic visualizations are built with components that bring useful features.

  • Annotations
  • Confidence band
  • Legend
  • Color palettes

Annotations

Try double clicking the annotations to move them.

Static threshold

var annotationsScatterplot = new proteic.Scatterplot(data,{
    xAxisLabel:'Petal length (cm)',
    yAxisLabel:'Petal width (cm)',
    markerShape: 'star',
    legendPosition: 'bottom',
})
.annotations([
    { type: 'threshold', axis: 'y', value: 1.8, text: 'Mean width' },
    { type: 'threshold', axis: 'x', value: 2, text: 'Mean length' }
]);
annotationsScatterplot.draw();

Dynamic threshold

var chart = new proteic.Linechart(data,{...})
.annotations([
    { type: 'threshold', axis: 'y', variable: 'flow', text: 'current flow'}
])
.datasource(ds)
chart.draw();

Dynamic error band

var chart = new proteic.Linechart(data,{...})
.annotations([
    { type: 'threshold', axis: 'y', variable: 'flow', text: 'current flow'},
    { type: 'band', axis: 'y',
      variable: 'flow', width: 'flowError', text: 'error'}
])
.datasource(ds)
chart.draw();

Alerts

var chart = new proteic.Linechart(data,{...})
.annotations([
    { type: 'threshold', axis: 'y', value: 0.1, text: 'rain alert' },
])
.alert('rain', (value, events) => {
    return  value > 0.1;
}, callback, events)
.datasource(ds)
chart.draw();

Confidence Band

The confidence band can be defined

User-defined Confidence

var chart = new proteic.Linechart(data,{
    confidenceBandOpacity: 0.3,
    confidenceModifier: (confidence) => 0.5 * confidence,
    propertyError: 'flowError'
}).unpivot(['rain', 'flow'])
.datasource(ds);
chart.draw();
